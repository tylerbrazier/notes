Github actions
==============

[workflows](https://docs.github.com/en/actions/concepts/workflows-and-actions/workflows)
are made up of
[actions](https://docs.github.com/en/actions/concepts/workflows-and-actions/custom-actions)

    .github/
    ├── actions
    │   └── do-small-task
    │       └── action.yml
    └── workflows
        ├── build.yml
        └── deploy.yml

- [workflow yml reference](https://docs.github.com/en/actions/reference/workflows-and-actions/workflow-syntax)
- [action yml reference](https://docs.github.com/en/actions/reference/workflows-and-actions/metadata-syntax)
